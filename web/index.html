Get Favicon Service

Use this service to get an icon of a specified webpage. It tries to retrieve all available icons from the webpage not only favicon to get the result. You need to use GET method, pass url as a path (possible, with http or https protocol) and can specify optional parameters.

Example: 
    /www.google.ru
    /https://www.google.ru/?size=128&format=png
Parameters:
sizeThe size of the image in pixels, 16 dy default
formatThe format of the response, can be one of: png, ico, gif, bmp, jpg or jpeg, default is png.
